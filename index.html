BookLooker - In Production

An Android application created to allow used-book merchants to quickly price-check books while they are out expanding their inventory. Features ISBN input, barcode scanning, and title search for books published before the introduction of ISBNs. The application will search a number of online book-selling marketplaces and return the price of the book on each, as well as a link should the user decide to sell the book on that store.

Bagwork Boxing Tool - In Production

A mobile application that automates calling out combinations during heavy-bag work.

GorillaJelly - In Production

GorillaJelly is a Windows stop-motion animation application. Created primarily for DSLR image-capture and playback, it also features onion-skinning, letterbox masking for frame composition, and audio playback for syncing animation to audio files. In the future, I plan to add more frame composition tools, output format options, and a quick shot-preview function. The program will also be migrated to a more modern user interface at some point. Free for personal use!
